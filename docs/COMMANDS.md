# Supported Voice Commands

## Navigation Commands
- "Navigate to [location]"
- "Find route to [location]"
- "Avoid obstacles"
- "Where am I?"

## Object Recognition
- "What objects are around me?"
- "Detect objects"
- "Stop object detection"

## Face Recognition
- "Who is this?"
- "Add new face [name]"
- "Recognize faces"
- "Stop face recognition"

## Currency Recognition
- "What currency is this?"
- "Detect currency"
- "Stop currency detection"

## System Control
- "Exit"
- "Quit"
- "Help"
